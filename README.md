# Surya Painting Web Dashboard

Next.js frontend untuk Surya Painting management system.

## Setup

```bash
npm install
npm run dev
```

Visit http://localhost:3000

## Deployment

```bash
vercel deploy --prod --project suryapainting-web-app
```

## Environment

- `NEXT_PUBLIC_API_URL` - Backend API URL (set in `.env.local`)

Default: https://suryapaintingapp.vercel.app
